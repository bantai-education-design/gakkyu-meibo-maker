## ソフトの概要

学級名簿メーカーは、学校の先生向けに、学級名簿、提出物チェック表、保護者会受付名簿、健康観察表、集金確認表、班別名簿、誕生日一覧などを作成・印刷できる無料ソフトです。

画面で名簿の形を整え、印刷プレビューで確認しながら、そのまま印刷しやすい表を作れます。

## 主な機能

- 学級名簿の作成
- 提出物チェック表の作成
- 保護者会受付名簿の作成
- 健康観察表、集金確認表、班別名簿、誕生日一覧の作成
- CSV読み込み
- Excelなどの表からのコピー＆貼り付け取り込み
- 手入力による名簿作成
- 漢字＋ふりがな、ひらがな、漢字のみの氏名表記切り替え
- 1年生向けのひらがな名簿
- JSONファイルによる保存・読み込み
- 印刷プレビュー

## 動作環境

- Windows 10 / Windows 11
- 64bit環境推奨

## インストール方法

1. ZIPファイルを展開します。
2. 展開したフォルダー内の `ClassRosterMaker-0.6.5-x64.exe` を起動します。

インストール作業は不要です。

## アンインストール方法

展開したフォルダーを削除してください。

共有パソコンで使用した場合は、保存データやダウンロードしたファイルが残っていないか確認してください。

## 個人情報の取り扱い

実名の児童データを扱う場合は、保存ファイルの取り扱いに注意してください。
実名入りのCSVや保存ファイルを公開場所にアップロードしないでください。
共有パソコンで使用した場合は、使用後に保存データやダウンロードしたファイルの保管場所を確認してください。
学校や自治体の個人情報の取り扱いルールに従って使用してください。

## 印刷時の注意

きれいに印刷するには、印刷画面で「ヘッダーとフッター」をOFF、「背景のグラフィック」をONにしてください。
余白は「なし」または「最小」がおすすめです。

プリンターやWindows側の設定によって、印刷画面の表示や余白の扱いが異なる場合があります。
