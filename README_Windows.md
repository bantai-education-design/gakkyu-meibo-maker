# 学級名簿メーカー Windows版メモ

## 起動方法

配布された `学級名簿メーカー-バージョン-win.exe` をダブルクリックして起動します。
初回起動時に Windows の確認画面が出る場合は、配布元を確認してから実行してください。

## 使い方の概要

- 左側の操作パネルで、用途テンプレート、所属、タイトル、氏名表記、レイアウトを設定します。
- 右側の印刷プレビューで、印刷される表の見た目を確認します。
- 「データ編集」画面では、児童データを一覧表で編集できます。
- CSV読み込み、Excel風の貼り付け入力、JSON保存・読み込みを利用できます。

## JSON保存・読み込み

現在の名簿データと設定は、JSONファイルとして保存できます。
保存したJSONファイルを読み込むと、児童データ、用途テンプレート、所属、タイトル、レイアウト設定などを復元できます。

## 個人情報の取り扱い

実名の児童データを扱う場合は、保存ファイルの取り扱いに注意してください。
実名入りのCSVやJSONをGitHub等にアップロードしないでください。
共有PCで使用した場合は、使用後に保存データやダウンロードしたファイルの保管場所を確認してください。

## 印刷時の注意

きれいに印刷するには、印刷画面で「ヘッダーとフッター」をOFF、「背景のグラフィック」をONにしてください。
余白は「なし」または「最小」がおすすめです。

Electron版でも通常の印刷操作は `window.print()` を使います。プリンターやWindows側の設定によって、印刷画面の表示や余白の扱いが異なる場合があります。
