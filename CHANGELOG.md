# 変更履歴

## v0.6.3 - 2026-05-26

- Windows配布用に `npm run dist` を追加し、electron-builderでポータブル版exeを作成できる構成に変更
- `electron:build` はunpacked形式の確認用、`dist` は配布物作成用としてnpm scriptsを整理
- Windows用のartifact名を `${productName}-${version}-${arch}.${ext}` に設定
- `README_Windows.md` を追加し、Windows版の起動方法、個人情報の取り扱い、JSON保存、印刷時の注意を記載

## v0.6.2 - 2026-05-26

- 氏名表記「ひらがな」専用の文字数クラスを追加し、長いひらがな氏名でも右端で欠けにくい表示に調整
- ひらがな氏名の文字数に応じてフォントサイズ、字間、横方向の縮小率を段階的に切り替えるように変更
- ひらがな表示時だけ氏名欄を最小限広げ、チェック欄を優先するコンパクト方針を維持
- 印刷時もひらがな氏名が省略記号や右端欠けにならないようCSSを調整

## v0.6.1 - 2026-05-26

- 初期サンプル設定の所属を「1年1組」にし、印刷プレビューの帳票タイトルへ所属が入りやすい状態に調整
- タイトル生成時に `classLabel + title` の規則を明確化し、所属がある場合は全テンプレートでタイトル先頭に表示されるように確認
- 基本セクションの所属入力欄に「タイトルの先頭に表示されます。」の案内を追加
- テンプレート切り替え時に所属を保持したまま、通常名簿・提出物チェック表・誕生日一覧・班別名簿などへ反映されることを確認

## v0.6.0 - 2026-05-26

- Electron用の `electron/main.ts` と `electron/preload.ts` を追加
- `npm run electron:dev` でVite開発サーバーを読み込むElectronウィンドウを起動できるように変更
- build後に `dist/index.html` を読み込めるElectron単体化準備を追加
- `electron:compile` と `electron:build` を追加し、Windows版exe/installer化に向けた構成に整理
- nodeIntegrationをfalse、contextIsolationをtrueにし、最小限のpreload APIに抑えた安全設定に調整

## v0.5.13 - 2026-05-26

- 貼り付け取り込みで「現在の名簿に追加する」「現在の名簿を置き換える」を選べるようにした
- データ編集画面に「番号を振り直す」「空白行を削除」を追加
- 氏名未入力、ふりがな未入力、番号未入力、番号重複を一覧表で分かるように表示
- 貼り付け時の性別表記ゆれとして、男子、女子、男の子、女の子、m/M、f/Fを正規化
- データ編集画面上部の操作ボタンを整理

## v0.5.12 - 2026-05-26

- 印刷プレビューとデータ編集を切り替える表示モードを追加
- データ編集画面で児童一覧を表形式で直接編集できるようにした
- 1行追加、選択行削除、全データ消去を追加
- Excel風のタブ区切り表を貼り付けて取り込める貼り付け入力欄を追加
- 貼り付け時に既存のCSV列対応ロジックを利用し、番号、氏名、ふりがな、性別、班、備考などに対応

## v0.5.11 - 2026-05-26

- 班別名簿テンプレートでは班列表示とグループ順を維持
- 古い保存データでも表示列設定が欠けた場合に既定値で補正されるように調整
- 所属入力欄とプレビュータイトルの `所属 + タイトル` 表示を確認

## v0.5.10 - 2026-05-26

## v0.5.9 - 2026-05-26

## v0.5.8 - 2026-05-26

## v0.5.7 - 2026-05-26
